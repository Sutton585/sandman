# MISSION: Arbitrage over Competition
## The Operator Manifesto

### 1. The Core Strategy
The goal of this system is not to help the user "find a job" in the traditional sense. It is to achieve **Arbitrage over Competition**. We seek **Unknown Unknowns**— opportunities where the user's asymmetric technical skills (UX, DesignOps, AI Automation, Python) provide an unfair advantage over low-competition applicant pools.

### 2. Strategic Pillars
- **Silent Domination & Stacking:** We target roles that are remote, low-oversight, and automatable. The goal is to perform at a high level in a fraction of the time through silent automation, enabling the stacking of multiple income streams (the /r/overemployed model).
- **Non-Traditional Exploitation:** We seek "boring" industries (local businesses, SMBs, government) that are under-digitized. A technically sophisticated "Trojan Horse" entry allows for rapid modernization and potential acquisition/buyout paths.
- **AI as a Discovery Agent:** The system must act as a discovery engine, suggesting roles, industries, and niches the user would not normally consider. It looks for *signals of exploitability*.
- **Income Stacking & Diversification:** The project covers:
    - **W2 Traditional/Arbitrage Jobs:** Senior UX, DesignOps, or vague "Admin" roles.
    - **Government Micro-Contracts:** SAM.gov or local RFP contracts that can be fulfilled or middle-manned using AI.
    - **Surplus & Hardware Arbitrage:** Monitoring marketplaces for flippable hardware or LLM lab equipment.
    - **SMB Consultancy:** Selling modernization services to businesses in transition.

### 3. Core Principles
- **Arbitrage over Competition:** Value lies in the unexpected application of skills.
- **Human-in-the-Loop:** The system researches, evaluates, and drafts; the user judges and submits.
- **Honest Reframing, Not Fabrication:** Translation of real experience into employer-specific terminology.
- **Profile as Source of Truth:** A single high-fidelity document (`profile.md`) drives all agent output.
- **Unknown Unknowns Matter Most:** The most valuable opportunity is the one the user hasn't thought to search for.

### 4. The Problem
Standard job searching is manual, inefficient, and optimized for average candidates. It ignores the universe of non-obvious opportunities that require research patterns humans aren't built for, but machines excel at.
